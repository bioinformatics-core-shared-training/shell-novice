Downloading Files

wget utility is the best option to download files from internet. It can pretty much handle all complex download situations including large file downloads, recursive downloads, non-interactive downloads, multiple file downloads etc. It retrieves files from World Wide Web (WWW) using widely used protocols like HTTP, HTTPS and FTP. It is designed in such way so that it works in slow or unstable network connections. Wget automatically start download where it was left off in case of network problem. Also downloads file recursively. Itâ€™ll keep trying until file has be retrieved completely.

The command wget will download single file and stores in a current directory. It also shows download progress, size, date and time while downloading.

Accessing Remote Server

ssh (Secure Shell) is a network protocol that allows a secure access over an encrypted connection. Through an SSH connection you can easily manage your files and folders, modify their permissions, edit files directly on the server, configure and install your scripts, etc. ssh is used to securely login to a Linux / UNIX host running the sshd daemon on a reachable network.

We will be accessing our hpcgate server using our CRUK-CI credentials and ssh command. Below is the command to use, but you will have to replace pajon01 by your personal username. Then you will have to type yes to the question Are you sure you want to continue connecting (yes/no)? and finally enter your own password to get in.

$ ssh pajon01@hpcgate.cri.camres.org
The authenticity of host 'hpcgate.cri.camres.org (10.20.192.21)' can't be established.
RSA key fingerprint is 9b:1b:7c:8e:64:d8:bd:ae:0b:f2:8c:f3:60:87:30:b9.
Are you sure you want to continue connecting (yes/no)? yes
Warning: Permanently added 'hpcgate.cri.camres.org,10.20.192.21' (RSA) to the list of known hosts.
pajon01@hpcgate.cri.camres.org's password:
Last login: Wed Apr 15 10:52:59 2015 from 10.20.14.45

At this point, you are physically on the hpcgate server inside your home directory. If you now type these commands the outputs will now be specific to you.

$ whoami
pajon01
$ pwd
/home/pajon01
$ ls

To return to your own machine as user nelle, you will have to type the exit command.

$ exit

Copying Files and Directories

ssh protocol can also be used to copy files & directories, using the same connection method as above but the command we use is scp.

This will copy the file notes.txt from its current location on your computer to your home directory on hpcgate. 

$ scp notes.txt pajon01@hpcgate.cri.camres.org:
